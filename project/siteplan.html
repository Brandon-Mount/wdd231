Cooking Compass – Site Plan

Site Name

Cooking Compass

This name symbolizes a helpful guide for navigating the world of cooking. Like a compass provides direction, this site will point users toward great recipes, essential tips, and proper use of cooking tools.

Optional domain availability: cookingcompass.net

Site Purpose

Cooking Compass is designed to help beginner and intermediate home cooks improve their culinary skills. The website provides bite-sized cooking facts and tips, detailed recipes, and an interactive guide to kitchen tools and how to use them properly.

Scenarios

  • What’s the easiest way to sauté vegetables without burning them?
  • Which cooking tools do I actually need as a beginner?

Color Schema

  • Accent Color: #ff6f3c – Used for headings, call-to-action buttons, and highlights.
  • Background Color: #fff7ed – Used for the overall page background to give a warm, kitchen-like feel.

Typography

  • Georgia – Used for headings to add a touch of classic, refined style.
  • Segoe UI – Used for body text to ensure readability and a modern look.

Wireframe

Mobile View:

  • Top: Logo and site name
  • Middle: Vertical stack – Tips section, Recipe preview section, Tool overview section
  • Bottom: Navigation menu bar (Home, Recipes, Tools)

Desktop View:

  • Top: Logo aligned left, Nav menu right (Home, Recipes, Tools)
  • Main Body: Grid layout – left sidebar for tips, center column for featured recipes, right column for tool spotlight
  • Footer: Contact and social media links
